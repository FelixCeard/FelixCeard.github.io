Hi, I'm Felix Ceard-Falkenberg
About me

I am currently a student at the Saarland University studying computer science with a strong focus on artificial intelligence and data science.

I am also currently working as a research assistant at the German Research Center for Artificial Intelligence (DFKI) in the Interactive Machine Learning (IML) department.

Besides computer science, I have several other interests, such as music.


Projects
Programming language

Eb-maj7 is my own esoteric programming language.

The goal of this programming language was to be able to write functional code that would generate music during execution.

Neural Network framework

After being introduced to functional programming languages in programming 1, I have challenged myself to write a (small) framework for the training of neural networks.
This meant implementing gradient descent from scratch.

Transformer

For the course High Level Computer Vision, I am doing a project involving training a large transformer from scratch. The model I am training has 27 million parameter.
The goal of this transformer is to be able to recreate pictures of objects solely on their sketches.
As it is work in progress, not a lot can be published yet.
